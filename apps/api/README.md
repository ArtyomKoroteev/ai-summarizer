# API (NestJS)

Backend API built with [NestJS](https://nestjs.com/).

## Setup

From the **project root**, install dependencies:

```bash
pnpm install
```

## Development

From the **project root**:

```bash
pnpm --filter api dev
```

Or from this directory:

```bash
pnpm dev
```

The API will run on `http://localhost:3000` by default.

## Build

```bash
pnpm build
```

## Testing

```bash
# Unit tests
pnpm test

# E2E tests
pnpm test:e2e

# Test coverage
pnpm test:cov
```

## Production

```bash
pnpm start:prod
```
